## MODIFIED Requirements

### Requirement: 单页翻页阅读
系统 SHALL 以单页模式展示漫画图片，每次显示一张图片。翻页方向由当前书籍的阅读方向设置决定。RTL（从右到左）模式下，键盘右方向键翻到上一页、左方向键翻到下一页，点击屏幕右侧翻到上一页、左侧翻到下一页。LTR（从左到右）模式下，行为与传统一致。

#### Scenario: LTR 模式下使用键盘右方向键翻到下一页
- **WHEN** 当前阅读方向为 LTR，用户按下键盘右方向键（→）且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: LTR 模式下使用键盘左方向键翻到上一页
- **WHEN** 当前阅读方向为 LTR，用户按下键盘左方向键（←）且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: RTL 模式下使用键盘左方向键翻到下一页
- **WHEN** 当前阅读方向为 RTL，用户按下键盘左方向键（←）且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: RTL 模式下使用键盘右方向键翻到上一页
- **WHEN** 当前阅读方向为 RTL，用户按下键盘右方向键（→）且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: LTR 模式下点击屏幕右侧翻到下一页
- **WHEN** 当前阅读方向为 LTR，用户点击图片区域右侧三分之一区域且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: LTR 模式下点击屏幕左侧翻到上一页
- **WHEN** 当前阅读方向为 LTR，用户点击图片区域左侧三分之一区域且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: RTL 模式下点击屏幕左侧翻到下一页
- **WHEN** 当前阅读方向为 RTL，用户点击图片区域左侧三分之一区域且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: RTL 模式下点击屏幕右侧翻到上一页
- **WHEN** 当前阅读方向为 RTL，用户点击图片区域右侧三分之一区域且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: 在最后一页无法继续前进
- **WHEN** 用户已是最后一页，执行「下一页」操作（无论方向）
- **THEN** 系统保持在当前页，不执行任何操作

#### Scenario: 在第一页无法继续后退
- **WHEN** 用户已是第一页，执行「上一页」操作（无论方向）
- **THEN** 系统保持在当前页，不执行任何操作

### Requirement: 连续滚动阅读模式
系统 SHALL 提供连续滚动阅读模式，将所有页面纵向排列在一个可滚动容器中，用户可自由上下滚动浏览。阅读方向设置影响键盘左右方向键的映射（RTL 时 ← 向前滚动、→ 向后滚动），但不影响纵向滚动行为和鼠标滚轮方向。

#### Scenario: 切换到连续滚动模式
- **WHEN** 用户通过设置面板选择"连续滚动"阅读模式
- **THEN** 系统将页面切换为纵向连续排列的滚动视图

#### Scenario: LTR 模式下连续滚动的键盘操作
- **WHEN** 当前阅读方向为 LTR，用户按下 → 或 ↓
- **THEN** 系统向下滚动（向前翻页）

#### Scenario: RTL 模式下连续滚动的键盘操作
- **WHEN** 当前阅读方向为 RTL，用户按下 ← 或 ↓
- **THEN** 系统向下滚动（向前翻页）

#### Scenario: 连续滚动模式下的页面显示
- **WHEN** 用户在连续滚动模式下浏览漫画
- **THEN** 页面图片宽度填满阅读区域，保持原始宽高比，页面之间无间距或仅有极小间距

#### Scenario: 连续滚动模式下的进度保存
- **WHEN** 用户在连续滚动模式下滚动浏览
- **THEN** 系统以可视区域中心对应的页面作为当前页，自动保存阅读进度

#### Scenario: 从连续滚动模式切回单页模式
- **WHEN** 用户从连续滚动模式切换到单页模式
- **THEN** 系统跳转到连续滚动中最后浏览的页面，恢复单页显示

#### Scenario: 连续滚动模式下的内存控制
- **WHEN** 漫画页数较多（超过 50 页）时
- **THEN** 系统仅渲染可视区域及上下各 2 页 buffer 的图片，其余位置显示占位区域

### Requirement: 阅读模式切换
系统 SHALL 在阅读器工具栏提供齿轮设置按钮，点击弹出设置面板，面板内可切换阅读模式和阅读方向。工具栏和页码滑条在无鼠标操作 3 秒后自动隐藏，鼠标移至屏幕顶部或底部 40px 区域时浮现。

#### Scenario: 显示齿轮设置按钮
- **WHEN** 用户在阅读器中浏览漫画
- **THEN** 顶部工具栏右侧显示齿轮图标按钮

#### Scenario: 点击齿轮按钮弹出设置面板
- **WHEN** 用户点击齿轮按钮
- **THEN** 系统在齿轮按钮下方弹出设置面板，显示阅读模式和阅读方向选项，当前值高亮标记

#### Scenario: 在设置面板中切换阅读模式
- **WHEN** 用户在设置面板中点击非当前阅读模式的选项
- **THEN** 系统立即切换到选中的阅读模式，保持当前阅读位置不变，面板保持打开

#### Scenario: 点击面板外部关闭面板
- **WHEN** 设置面板打开时，用户点击面板外部区域
- **THEN** 面板关闭

#### Scenario: 按 ESC 关闭面板
- **WHEN** 设置面板打开时，用户按下 ESC 键
- **THEN** 面板关闭（不触发阅读器返回书库行为）

#### Scenario: 无操作时 UI 自动隐藏
- **WHEN** 用户在阅读器中 3 秒内未移动鼠标
- **THEN** 工具栏和页码滑条通过过渡动画自动隐藏

#### Scenario: 鼠标移至底部区域时页码滑条浮现
- **WHEN** 沉浸模式下，用户将鼠标移至屏幕底部 40px 区域内
- **THEN** 底部页码滑条以过渡动画浮现

#### Scenario: 操作 UI 时不触发隐藏
- **WHEN** 用户正在操作工具栏或页码滑条
- **THEN** UI 保持可见，不触发自动隐藏计时
