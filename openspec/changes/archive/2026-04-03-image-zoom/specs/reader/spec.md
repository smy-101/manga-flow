## MODIFIED Requirements

### Requirement: 单页翻页阅读
系统 SHALL 以单页模式展示漫画图片，每次显示一张图片。翻页方向由当前书籍的阅读方向设置决定。RTL（从右到左）模式下，键盘右方向键翻到上一页、左方向键翻到下一页，点击屏幕右侧翻到上一页、左侧翻到下一页。LTR（从左到右）模式下，行为与传统一致。当缩放比例 > 100% 时，禁用点击翻页。

#### Scenario: LTR 模式下使用键盘右方向键翻到下一页
- **WHEN** 当前阅读方向为 LTR，用户按下键盘右方向键（→）且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: LTR 模式下使用键盘左方向键翻到上一页
- **WHEN** 当前阅读方向为 LTR，用户按下键盘左方向键（←）且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: RTL 模式下使用键盘左方向键翻到下一页
- **WHEN** 当前阅读方向为 RTL，用户按下键盘左方向键（←）且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: RTL 模式下使用键盘右方向键翻到上一页
- **WHEN** 当前阅读方向为 RTL，用户按下键盘右方向键（→）且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: LTR 模式下点击屏幕右侧翻到下一页
- **WHEN** 当前阅读方向为 LTR，缩放比例为 100%，用户点击图片区域右侧三分之一区域且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: LTR 模式下点击屏幕左侧翻到上一页
- **WHEN** 当前阅读方向为 LTR，缩放比例为 100%，用户点击图片区域左侧三分之一区域且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: RTL 模式下点击屏幕左侧翻到下一页
- **WHEN** 当前阅读方向为 RTL，缩放比例为 100%，用户点击图片区域左侧三分之一区域且当前不是最后一页
- **THEN** 系统切换到下一页图片

#### Scenario: RTL 模式下点击屏幕右侧翻到上一页
- **WHEN** 当前阅读方向为 RTL，缩放比例为 100%，用户点击图片区域右侧三分之一区域且当前不是第一页
- **THEN** 系统切换到上一页图片

#### Scenario: 在最后一页无法继续前进
- **WHEN** 用户已是最后一页，执行「下一页」操作（无论方向）
- **THEN** 系统保持在当前页，不执行任何操作

#### Scenario: 在第一页无法继续后退
- **WHEN** 用户已是第一页，执行「上一页」操作（无论方向）
- **THEN** 系统保持在当前页，不执行任何操作

#### Scenario: 放大状态下点击不翻页
- **WHEN** 缩放比例 > 100%，用户点击图片区域的左侧或右侧
- **THEN** 系统不执行翻页操作

### Requirement: 阅读器设置面板
系统 SHALL 在阅读器工具栏提供齿轮设置按钮，点击弹出设置面板，面板内可调整阅读模式、阅读方向和适配模式。设置面板中的修改即时生效并自动持久化。工具栏右侧提供 Chrome 风格缩放控件（`[-] N% [+]`）。

#### Scenario: 阅读器设置面板内容
- **WHEN** 用户打开阅读器设置面板
- **THEN** 面板显示三组设置：「阅读模式」（单页模式/连续滚动/双页展开）、「阅读方向」（从左到右/从右到左）和「适配模式」（最佳适配/适配宽度/适配高度/原始大小），当前值标记为选中状态

#### Scenario: 面板中选择适配模式
- **WHEN** 用户在设置面板中点击非当前适配模式的选项
- **THEN** 系统立即切换适配模式，更新阅读器图片显示，将新模式写入 `book_preferences`

#### Scenario: 连续滚动模式下适配模式选项灰显
- **WHEN** 当前阅读模式为连续滚动
- **THEN** 适配模式选项组灰显，提示「连续滚动模式下不可用」

#### Scenario: 面板中选择阅读模式为双页展开
- **WHEN** 用户在设置面板中点击「双页展开」选项
- **THEN** 系统立即切换到双页展开视图，保持当前阅读位置（调整到最近的 spread 起始页）

#### Scenario: 面板中选择阅读方向
- **WHEN** 用户在设置面板中点击非当前阅读方向的选项
- **THEN** 系统立即切换阅读方向，更新阅读器翻页行为，将新方向写入 `book_preferences`

#### Scenario: 点击面板外部关闭面板
- **WHEN** 设置面板打开时，用户点击面板外部区域
- **THEN** 面板关闭

#### Scenario: 按 ESC 关闭面板
- **WHEN** 设置面板打开时，用户按下 ESC 键
- **THEN** 面板关闭（不触发阅读器返回书库行为）

#### Scenario: 无操作时 UI 自动隐藏
- **WHEN** 用户在阅读器中 3 秒内未移动鼠标
- **THEN** 工具栏和页码滑条通过过渡动画自动隐藏

#### Scenario: 鼠标移至底部区域时页码滑条浮现
- **WHEN** 沉浸模式下，用户将鼠标移至屏幕底部 40px 区域内
- **THEN** 底部页码滑条以过渡动画浮现

#### Scenario: 操作 UI 时不触发隐藏
- **WHEN** 用户正在操作工具栏或页码滑条
- **THEN** UI 保持可见，不触发自动隐藏计时

#### Scenario: 工具栏显示缩放控件
- **WHEN** 用户在阅读器中浏览漫画
- **THEN** 工具栏右侧显示缩放控件 `[-] N% [+]`，N 为当前缩放百分比

#### Scenario: 点击缩放控件加号按钮
- **WHEN** 用户点击缩放控件的 `[+]` 按钮，且当前缩放比例 < 500%
- **THEN** 系统将缩放比例增加 25%

#### Scenario: 点击缩放控件减号按钮
- **WHEN** 用户点击缩放控件的 `[-]` 按钮，且当前缩放比例 > 25%
- **THEN** 系统将缩放比例减少 25%

#### Scenario: 点击缩放控件百分比文字重置
- **WHEN** 当前缩放比例不为 100%，用户点击百分比文字
- **THEN** 系统将缩放比例重置为 100%
