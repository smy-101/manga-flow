## MODIFIED Requirements

### Requirement: 阅读模式切换
系统 SHALL 在阅读器工具栏提供模式切换按钮，允许用户在单页模式和连续滚动模式之间切换。工具栏和页码滑条在无鼠标操作 3 秒后自动隐藏，鼠标移至屏幕顶部或底部 40px 区域时浮现。

#### Scenario: 显示模式切换按钮
- **WHEN** 用户在阅读器中浏览漫画
- **THEN** 顶部工具栏显示阅读模式切换按钮，当前模式高亮显示；工具栏在无操作 3 秒后自动隐藏，鼠标移至屏幕顶部 40px 区域时浮现

#### Scenario: 点击切换按钮切换模式
- **WHEN** 用户点击阅读模式切换按钮
- **THEN** 系统在单页模式和连续滚动模式之间切换，保持当前阅读位置不变

#### Scenario: 无操作时 UI 自动隐藏
- **WHEN** 用户在阅读器中 3 秒内未移动鼠标
- **THEN** 工具栏和页码滑条通过过渡动画自动隐藏

#### Scenario: 鼠标移至底部区域时页码滑条浮现
- **WHEN** 沉浸模式下，用户将鼠标移至屏幕底部 40px 区域内
- **THEN** 底部页码滑条以过渡动画浮现

#### Scenario: 操作 UI 时不触发隐藏
- **WHEN** 用户正在操作工具栏或页码滑条
- **THEN** UI 保持可见，不触发自动隐藏计时
