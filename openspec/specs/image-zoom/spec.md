# Image Zoom

## Purpose

图片缩放交互：允许用户在阅读器中对漫画图片进行自由缩放、双击快速缩放、放大后拖拽平移，并在工具栏和连续滚动模式中提供缩放支持。

## Requirements

### Requirement: 图片缩放交互
系统 SHALL 允许用户在当前 FitMode 基础上对图片进行自由缩放。缩放范围 25%（0.25x）至 500%（5.0x），步进 25%。缩放状态为会话临时状态，不持久化。

#### Scenario: Ctrl + 滚轮放大
- **WHEN** 用户在阅读区域按住 Ctrl 键并向上滚动鼠标滚轮，且当前缩放比例 < 500%
- **THEN** 系统将缩放比例增加 25%（如从 100% 变为 125%）

#### Scenario: Ctrl + 滚轮缩小
- **WHEN** 用户在阅读区域按住 Ctrl 键并向下滚动鼠标滚轮，且当前缩放比例 > 25%
- **THEN** 系统将缩放比例减少 25%（如从 100% 变为 75%）

#### Scenario: 缩放达到上限
- **WHEN** 当前缩放比例已为 500%，用户执行放大操作
- **THEN** 系统保持 500% 不变，不继续放大

#### Scenario: 缩放达到下限
- **WHEN** 当前缩放比例已为 25%，用户执行缩小操作
- **THEN** 系统保持 25% 不变，不继续缩小

#### Scenario: Ctrl + 加号步进放大
- **WHEN** 用户按下 Ctrl + =（或 Ctrl + +），且当前缩放比例 < 500%
- **THEN** 系统将缩放比例增加 25%

#### Scenario: Ctrl + 减号步进缩小
- **WHEN** 用户按下 Ctrl + -，且当前缩放比例 > 25%
- **THEN** 系统将缩放比例减少 25%

#### Scenario: Ctrl + 0 重置缩放
- **WHEN** 用户按下 Ctrl + 0
- **THEN** 系统将缩放比例重置为 100%（即 FitMode 默认状态）

### Requirement: 双击快速缩放
系统 SHALL 支持双击阅读区域在 200% 放大与恢复之间切换。

#### Scenario: 未缩放时双击放大到 200%
- **WHEN** 当前缩放比例为 100%，用户双击阅读区域
- **THEN** 系统将缩放比例设为 200%

#### Scenario: 已缩放时双击恢复
- **WHEN** 当前缩放比例不为 100%，用户双击阅读区域
- **THEN** 系统将缩放比例恢复为 100%

### Requirement: 放大后拖拽平移
系统 SHALL 在图片放大后（缩放比例 > 100%）支持拖拽平移查看图片不同区域。放大状态下禁用点击翻页区域。

#### Scenario: 放大后拖拽平移
- **WHEN** 缩放比例 > 100%，用户在图片区域按住鼠标拖拽
- **THEN** 系统平移图片显示区域，用户可查看被裁剪的部分

#### Scenario: 放大状态下点击不翻页
- **WHEN** 缩放比例 > 100%，用户点击图片区域的左侧或右侧
- **THEN** 系统不执行翻页操作

#### Scenario: 恢复 100% 后重新启用点击翻页
- **WHEN** 缩放比例恢复为 100%，用户点击图片区域的左侧或右侧
- **THEN** 系统恢复正常的点击翻页行为

### Requirement: 工具栏缩放控件
系统 SHALL 在 ReaderToolbar 提供 Chrome 风格缩放控件，包含减号按钮、当前百分比显示、加号按钮，格式为 `[-] N% [+]`。

#### Scenario: 显示缩放控件
- **WHEN** 用户在阅读器中浏览漫画
- **THEN** 工具栏右侧显示缩放控件 `[-] 100% [+]`

#### Scenario: 点击加号按钮放大
- **WHEN** 用户点击缩放控件的 `[+]` 按钮，且当前缩放比例 < 500%
- **THEN** 系统将缩放比例增加 25%

#### Scenario: 点击减号按钮缩小
- **WHEN** 用户点击缩放控件的 `[-]` 按钮，且当前缩放比例 > 25%
- **THEN** 系统将缩放比例减少 25%

#### Scenario: 点击百分比重置缩放
- **WHEN** 当前缩放比例不为 100%，用户点击百分比文字
- **THEN** 系统将缩放比例重置为 100%

#### Scenario: 缩放比例更新时控件同步
- **WHEN** 用户通过任何方式（滚轮/键盘/双击）改变缩放比例
- **THEN** 工具栏缩放控件的百分比显示实时更新

### Requirement: 连续滚动模式缩放
系统 SHALL 在连续滚动阅读模式下支持缩放，缩放叠加在页面图片之上。

#### Scenario: 连续滚动模式下 Ctrl + 滚轮缩放
- **WHEN** 用户在连续滚动模式中按住 Ctrl 并滚动鼠标滚轮
- **THEN** 系统缩放所有页面图片，占位区域高度同步调整

#### Scenario: 缩放后连续滚动的虚拟滚动正确性
- **WHEN** 用户在连续滚动模式放大后滚动浏览
- **THEN** 系统正确渲染可视区域及 buffer 范围内的页面，未渲染区域显示高度正确的占位区域
